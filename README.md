# ProjectAuto: Relay Defense

세로 고정 모바일 플레이를 목표로 만든 협동 타워디펜스 프로토타입입니다. 웹/PWA로 바로 실행되며, iOS/Android 래핑 또는 Godot 이식 전의 플레이 검증용 기준 빌드로 사용할 수 있습니다.

## 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 플레이

- 기본은 봇 협동 모드입니다. 서버만 켜면 혼자서도 `AUTO-BOT`과 플레이할 수 있습니다.
- `Online`을 누르면 같은 서버의 WebSocket 룸에 접속합니다.
- 같은 네트워크의 다른 기기에서 `http://<서버IP>:4173`으로 접속하면 같은 세션을 공유합니다.
- 타워 버튼을 고른 뒤 빛나는 패드를 터치하면 건설합니다.
- 타워를 터치한 뒤 `Upgrade`로 협동 자원을 써서 강화합니다.
- `BM` 탭에는 현재 빌드에서 동작하는 상점, 일일 미션, 패스 구조가 들어 있습니다.

## 서버 배포 메모

Amazon EC2 같은 서버에서 Node.js 20 이상을 설치한 뒤 이 repo를 올리고 아래처럼 실행하면 됩니다.

```bash
PORT=4173 npm run start
```

보안 그룹에서 TCP `4173`을 열면 외부 기기에서 접속할 수 있습니다. HTTPS와 도메인이 붙으면 PWA 설치 경험과 iOS 권한 흐름이 더 좋아집니다.

## 구조

- `src/shared/content.js`: 레벨 디자인, 타워, 적, BM 아이템/패스/미션 데이터
- `src/shared/game.js`: 서버와 클라이언트가 공유하는 권위 게임 규칙
- `src/client/app.js`: 세로형 모바일 UI, 캔버스 그래픽, 봇/온라인 모드
- `server/server.js`: 정적 파일 서버와 단일 협동 WebSocket 룸
- `server/ws.js`: 의존성 없는 WebSocket 프레임 처리
- `test/`: 게임 규칙과 WebSocket 프레임 테스트

## 현재 완성 범위

- 세로 고정 390x720 게임 보드
- 2인 협동 자원 경제
- 온라인 WebSocket 멀티플레이
- 서버 없이 테스트 가능한 봇 협동 모드
- 2개 레벨 데이터와 웨이브 설계
- 3개 타워 역할군
- 런 내 상점, 재화, 패스, 미션 구조
- 절차적 캔버스 그래픽과 일관된 색채/타워 디자인
